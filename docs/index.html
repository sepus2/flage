Powerful, flexible and efficient templating engine for PHP. Compiled performance is very close to the same code written in php.

Basics

Flage statement starts with {: or {$ and ends with }
Ex: {: print_r(["{","}"],true) } OR {$var}

tags should be opened with {tag ...} and closed with {/tag}
{:if true} Valid {:else} Not valid {/if}

Supported values:

numbers : 1 , 1.0 , 1.0e-1

strings : "double quote", 'single quote\n'

booleans: true, false

null

functions : literal( named_argument = 'statement', 'positional argument')

tags : if, else, for, block, ...

variables : $var

Operators

Assignment (=, +=, -= )
{$var = 1}{$var} : 1
{$var+=2}{$var} : 3
Math (+, -, /, %, *, ++, --)
{$var = 1 + 1 * 2 }{$var} : 3
{$var2 = $var++}{$var} : 4
Logic (&&, ||)
{:if (true || false) && true}
Comparisons (==, !=, <, >, >=, <=, ===, !==)
{:if true==false || "1"===1}
Others (.+, |, ., [], (), ?:, \{)
{$var="a".+'b'}
{$var=["foo"="bar"]}
{$var.foo}
{$var['foo']}
\{ - will always output {

Functions

Any whitelisted php function is allowed.
To create Flage specific function can be created in plugins/functions/function_name.php, then create a function flage_function_[function_name]($flage,$arguments,$line,$col)
Where
$flage is the instance of Flage object
$arguments is an array of arguments (named or positioned)
$line is the line in the template file
$col is the column in template file
Example:
function flage_function_include($flage, $arguments, $line, $col){
    $parsedArgs=flage_generate_args(array('filename','inline'=>array('default'=>false)),$arguments,$line,$col);
    $filename = $parsedArgs['filename'];

And the call would be {:include("test.flg")} OR {:include(template="test.flg")}

Modifiers

Modifiers is a linux piping style of calling a function. A modifier can be any whitelisted php function, flage function or flage modifier in plugins/modifiers/modifier_name.php.
